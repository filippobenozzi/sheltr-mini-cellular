Sheltr Cloud

Sheltr Cloud is a lightweight MQTT + web control stack for DR154 devices. It lets you configure instances and control boards using Protocol 1.6 in transparent mode.

  • Public authenticated MQTT broker support.
  • Instance-based configuration for lights, shutters, dimmers, and thermostats.
  • Per-instance control pages with login, room grouping, status refresh, and hourly profiles.
  • Protocol frame publishing over MQTT with polling-based feedback.
